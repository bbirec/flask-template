Flask + React + PostgreSQL Template
-----------------------------------
Flask based web application template.

# Install
```bash
$ virtualenv venv
$ . venv/bin/activate
(venv)$ pip install -r requirements.txt
```

# Add/Remove deps
```bash
(venv)$ pip freeze > requirements.txt
```

# Run locally
```bash
(venv)$ python app.py
```
